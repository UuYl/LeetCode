# 709. To Lower Case

Implement function ToLowerCase() that has a string parameter str, and returns the same string in lowercase.

#### Example 1

> Input: "Hello"
> Output: "hello"

#### Example 2

> Input: "here"
> Input: "here"

#### Example 3

> Input: "LOVELY"
> Output: "lovely"
